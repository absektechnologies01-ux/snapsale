Project Description;project name : “ SnapSale”
I want to create a receipt generator app, it allows the shop owner to update items in the shop so when a customer buys from the shop, the shop owner selects the items bought, the app then list all the items in the receipt format and make it downloadable. The App allows shop owners to set shop name, Logo and necessary details and show preview before save, it should be editable, paper size should be receipt size(small).  record of items sold should be stored locally in a section of the app Dashboard view to show sales in a day, week and months. All this should be local. Use Flutter(IOS and Android) & Provider State Management  for this project. The UI should be nice with few animations,primary color: dark wine secondary : white
